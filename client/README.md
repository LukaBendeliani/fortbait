# Fortbait Client

A multiplayer battle royale game client built with Phaser 3 and Vite.

## Deploy to Vercel

1. Import this folder to Vercel
2. Set environment variables:
   - `VITE_SERVER_URL` — Your Railway server URL (e.g., `https://your-app.railway.app`)
   - `VITE_SERVER_PORT` — `443` for HTTPS

## Local Development

```bash
npm install
npm run dev
```

The client runs on http://localhost:3000

## Build

```bash
npm run build
```

Output: `dist/`
